The CSS Box Model

The CSS box model identifies the four parts of the HTML element. Let's consider each part of the box.

The four parts of the CSS box model

  1. Content
  2. Padding
  3. Border
  4. Margin

Each HTML element has a content box. This is the part of the element that contains the data and information. The content might be text, an image, or different HTML elements. The content box is just big enough to fit the content. You can change the size of the content box by using the CSS width and height properties along with different pixel sizes..

Around the content is the padding, border, and margin. The size of each of these can be set using CSS.
